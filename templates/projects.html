Blood Vessel Growth Simulator (2017)

Developed an internal simulation software to help Bioscience Researchers design experimental microfluidic devices (top part of the figure) and determine concentration of reagents (denoted v' on the bottom graph) required to stimulate the cells (denoted ρ'). This particular project involved replicating blood vessel growth in tumour environments. My MASc thesis consisted of: 1) preprocessing images into model verfication data, 2) building and implementing mathematical models that can identify the optimal configuration, and 3) deploying a simple user interface for reasearchers. Outcome of this project was reduction in time spent planning experiments by 40% and improvement in experiment success by 200%, which translated into savings of ~$20k/year for the lab.

Technologies: Matlab, Python

RetrofitAI (2019)

Built end-to-end pipeline capable of recommending relevant house renovations and estimating associated energy savings for homeowners as an alternative to traditional in-person energy audits. Trained an ML model to predict house consumptions and implemented specialized regression models to infer technical features from simple user inputs. Deployed models into production on AWS using Flask. Collaborated with the Product Team in creating a Minimum Viable Product (MVP) and with Engineering Team in creating an API to talk to the main company product.

Technologies: Python, Pandas, Scikit-Learn, XGBoost, S3, EC2, Flask, HTML, CSS, Bootstrap, MySQL

Customer Transaction Predictor (2020)

Utilized a Kaggle Santander Customer Data to build a model for predicting which customer will make a transaction irrespective of the transaction amount. Model built was a Decision Tree classifier and was deployed on AWS using Flask.

Technologies: Python, Scikit-Learn, PyCaret, AWS, Flask, HTML, CSS
